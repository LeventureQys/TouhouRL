# 东方Project强化学习项目

让AI自主玩东方Project弹幕射击游戏的强化学习系统。

## 项目简介

本项目使用强化学习技术训练AI玩东方Project系列STG游戏。通过计算机视觉识别游戏状态，结合DQN/PPO等算法，让AI学会躲避密集弹幕、击破敌机、收集道具等复杂操作。

**核心技术**：计算机视觉 + 轨迹预测 + 深度强化学习 + 课程学习

## 技术方案

### 视觉系统
通过OpenCV检测游戏元素：
- 自机位置和判定点
- 子弹位置、速度、大小
- 敌机和道具
- 预测子弹轨迹，生成危险度热力图

### 状态表示
使用特征向量（约165维）而非原始像素：
- 自机信息（位置、速度、生命值）
- 最近20个子弹（位置、速度、类型）
- 最近5个敌机
- 最近10个道具
- 8方向危险度评估

### 动作空间
6个按键的状态组合：
- 方向键：上、下、左、右
- Shift键：低速移动
- Bomb键：紧急回避

### 奖励函数
- 生存：+0.1/帧
- 死亡：-100
- 击破敌机：+1~100
- 收集道具：+1~5（上方收点更高）
- 擦弹：+0.3~0.5

### 训练策略
采用课程学习，从简单到复杂：
1. 简单场景：少量慢速子弹，学习基本躲避
2. 中等难度：加入敌机和道具，学习攻击和收集
3. 完整游戏：真实难度，学习高级技巧

## 核心挑战

1. **实时性**：60 FPS决策 → 轻量级网络 + GPU加速
2. **高维状态**：大量子弹 → 特征工程 + 注意力机制
3. **稀疏奖励**：密集奖励设计 + 课程学习
4. **泛化能力**：多样化训练 + 迁移学习
